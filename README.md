# React Weather App
:round_pushpin: Simple Weather App created with React library using Vite tool.

> visit : https://react-weather-app-vikas-parmar.vercel.app/

* Library : `React`
* Package Manager : `npm`
* Used Tool : `Vite`  
* Deployed On : `Vercel`

### Using Weather API for weather Data from -
> https://api.openweathermap.org

```
npm install
npm run dev
```
